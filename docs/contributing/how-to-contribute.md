# Como Contribuir

Obrigado pelo interesse em contribuir com o FinTrack!

1.  **Encontre uma Issue**: Olhe o quadro de Issues no GitHub. Comece pelas marcadas como "good first issue".
2.  **Fork o Repositório**: Crie sua cópia do projeto.
3.  **Prepare o Ambiente**: Siga o guia de [Setup](../development/setup-environment.md).
4.  **Crie uma Branch**: `feature/minha-contribuicao`.
5.  **Codifique**: Siga o [Guia de Estilo](../development/code-style.md).
6.  **Teste**: Garanta que nada quebrou.
7.  **Abra um PR**: Descreva o que você fez e referencie a Issue original.

## O que aceitamos?

- Correções de bugs.
- Melhorias de performance.
- Novas funcionalidades (discuta antes numa Issue).
- Melhorias na documentação.
